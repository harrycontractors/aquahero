# AquaHero Plumbing - Images Directory

## Logo Files

This directory should contain the following logo files:

### Required Logo Files:

1. **aquahero-icon.png** (Square Logo)
   - Size: 512x512px or larger
   - Use: Favicon, social media profile, app icon
   - Format: PNG with transparent background
   - Download from: [Logo Link 1](https://www.genspark.ai/api/files/s/1B0mHPVu)

2. **logo-full.jpg** (Complete Logo)
   - Size: 2000x1000px or similar
   - Use: Full branding, business cards, signage
   - Format: JPG or PNG
   - Download from: [Logo Link 2](https://www.genspark.ai/api/files/s/6qkVztk0)

3. **logo-header.jpg** (Horizontal Logo)
   - Size: 1365x768px or similar aspect ratio
   - Use: Website navigation bar, email signatures
   - Format: JPG or PNG
   - Download from: [Logo Link 3](https://www.genspark.ai/api/files/s/86zqzyqv)

## How to Add Your Logo Files

1. Download the logo files from the links above
2. Save them to this `images/` directory
3. Make sure the filenames match exactly:
   - `aquahero-icon.png`
   - `logo-full.jpg`
   - `logo-header.jpg`

## Optional Additional Images

You may also want to add:

## Image Optimization Tips

1. **Compress images** before uploading to reduce file size:
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target: <200KB for most images

2. **Use correct formats**:
   - JPG for photos
   - PNG for logos with transparency
   - WebP for modern browsers (optional)

3. **Responsive images**:
   - Provide multiple sizes for different devices
   - Use srcset attribute for better performance
